# PayLink

## Project Description

PayLink is a secure digital payment platform that enables users to send and receive money easily through a web or mobile interface while also supporting USSD transactions for users without internet access.

The platform integrates with trusted payment APIs and Mobile Money services to provide fast, secure, and accessible financial transactions.
